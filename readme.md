#NCKUcourseChecker.py
#DESCRIPTION
to check whether particular deparment has any course that is not full \(for NCKU only\)

#USAGE

##GUI
```shell
python3 GUI.py
```
and then input the deparment number

##Console mode
```shell
python3 NCKUcourseChecker.py
```
and then input the deparment number
